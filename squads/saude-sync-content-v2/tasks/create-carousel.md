# Tarefa: Criar Estrutura Completa do Carrossel

## Objetivo
Criar a estrutura completa de um carrossel Instagram com 8 a 10 slides. Para cada slide, definir conteúdo textual (headline + body copy) e especificações visuais completas para briefing ao Marco Mapa Visual.

## Input Esperado
- Tema central do carrossel
- Público-alvo (pacientes / médicos / cuidadores)
- Objetivo de negócio (educação / conversão / autoridade / retenção)
- Qualquer restrição temática ou de conteúdo

## Processo

### 1. Definir o Arco Narrativo
Antes de escrever qualquer slide, mapear o fluxo dramático:
- Qual é a tensão/problema de abertura?
- Qual é a virada (momento em que a solução aparece)?
- Qual é a prova/credibilidade?
- Qual é o CTA lógico?

### 2. Escrever Cada Slide

Para cada slide, entregar o bloco abaixo completo:

```
SLIDE [N] — [Tipo: Capa / Problema / Agravamento / Virada / Solução / Prova / CTA / Bônus]

HEADLINE: [máximo 8 palavras, Space Grotesk Bold 700]
BODY COPY: [máximo 3 linhas, DM Sans 500]
DADO DE DESTAQUE (se aplicável): [número ou estatística em JetBrains Mono 600]

COR DE FUNDO: [hex da paleta SaudeSync Dark]
COR DE ACENTO: [hex da paleta SaudeSync Dark]

BRIEFING VISUAL PARA MARCO:
- Tipo de imagem: [ilustração / foto / ícone / fundo / nenhuma]
- Função no slide: [o que a imagem deve comunicar no contexto deste slide]
- Emoção desejada: [como o usuário deve se sentir ao ver este slide]
- Composição: [onde a imagem aparece no slide e como interage com o texto]

NOTA DE LAYOUT: [qualquer instrução específica de design para este slide]
```

### 3. Verificar Coerência do Conjunto
Após criar todos os slides, revisar:
- O arco narrativo está completo e coerente?
- Nenhuma cor de fundo se repete por mais de 3 slides consecutivos?
- O CTA final é consequência lógica da narrativa?
- Todos os slides têm exatamente 1 ideia central?

## Output

Entregar os N blocos de slides completos, seguidos de um resumo do arco narrativo em 3 linhas.

## Critérios de Conclusão
- [ ] 8 a 10 slides criados
- [ ] Todos os campos de cada bloco preenchidos
- [ ] Paleta SaudeSync Dark usada corretamente em todos os slides
- [ ] Briefing visual completo para Marco em cada slide
- [ ] Arco narrativo identificável e coerente
- [ ] CTA final específico e acionável
