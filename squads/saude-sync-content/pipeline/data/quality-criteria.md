# Critérios de Qualidade — saude-sync-content

## Critérios Globais (Aplicados por Victor Veredicto)

### 1. Relevância para a Missão Saúde Sync (peso 20%)
- [ ] O conteúdo é consistente com "continuidade de cuidado"?
- [ ] O tom está alinhado com a palavra-essência CUIDADO?
- [ ] O conteúdo fortalece a confiança na marca?
- Threshold mínimo: 7/10

### 2. Adequação ao Público-Alvo (peso 20%)
- [ ] O espectro empático-técnico está calibrado para o público?
- [ ] A linguagem é compreensível para o leitor-alvo?
- [ ] A chamada de identificação está presente?
- Threshold mínimo: 7/10

### 3. Qualidade do Copy (peso 25%)
- [ ] O hook interrompe o scroll nos primeiros 3 segundos/caracteres?
- [ ] O corpo entrega o que o hook promete?
- [ ] Existe CTA claro e específico?
- [ ] Está dentro dos limites de caracteres por plataforma?
- Threshold mínimo: 7/10

### 4. Compliance Regulatório (peso 15%)
- [ ] Relatório da Carla Compliance aprovado (APROVADO ou APROVADO COM RESSALVAS)?
- [ ] Disclaimers obrigatórios incluídos?
- [ ] Nenhuma palavra proibida presente?
- Threshold: OBRIGATÓRIO — qualquer falha = REJECT automático

### 5. Qualidade Visual (peso 10%)
- [ ] Hierarquia visual clara (headline + suporte)?
- [ ] Contraste WCAG AA respeitado?
- [ ] Identidade Saúde Sync preservada (cores, fontes)?
- [ ] Slides sem poluição visual?
- Threshold mínimo: 7/10

### 6. Engajamento Potencial (peso 10%)
- [ ] O conteúdo é salvo? (save-worthy)
- [ ] O conteúdo é compartilhável? (quem mandaria para quem?)
- [ ] Existe fator de comentário? (o que levaria alguém a responder?)
- Threshold mínimo: 6/10

## Critérios Específicos por Formato

### Carrossel Instagram Feed
- [ ] 8-10 slides no total
- [ ] Slide 1 com hook poderoso (contradição, número, identificação)
- [ ] Cada slide: 40-80 palavras totais (headline + suporte)
- [ ] Cores alternadas entre slides (ritmo visual)
- [ ] Último slide com CTA específico
- [ ] Caption: primeiros 125 chars funcionam como hook independente

### Instagram Reels
- [ ] Duração: 15-30 segundos (ideal: 20-25s)
- [ ] Hook nos primeiros 2 segundos
- [ ] Legendas/subtítulos incluídos (conteúdo acessível sem som)
- [ ] Loop design (fim conecta ao início)
- [ ] Aspect ratio 9:16

### TikTok
- [ ] Duração: 30-60 segundos (ideal: 45s)
- [ ] Hook nos primeiros 3 segundos
- [ ] Dado mais surpreendente guardado para 30-38s
- [ ] Um único CTA
- [ ] Caption: máx. 100 chars
- [ ] Disclaimer de saúde incluído quando necessário

### Instagram Stories
- [ ] Sequência: 3-7 frames
- [ ] Texto por frame: máx. 3 linhas
- [ ] Pelo menos 1 elemento interativo (enquete, quiz, pergunta)
- [ ] Cada frame: consumível em 3-5 segundos

| Qualquer | Crítico < 4/10 | REJEITADO |

1. Compliance regulatório
2. Adequação ao público
3. Qualidade do copy (hook)
