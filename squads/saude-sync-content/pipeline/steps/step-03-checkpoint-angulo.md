# Checkpoint 3 — Seleção de Ângulo

O Igor Ideia gerou 3 perspectivas para o tema escolhido.

**O que você precisa fazer:**
Escolher **um dos 3 ângulos** para este ciclo de produção.

**Dica de escolha:**
